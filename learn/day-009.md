## Performance and stability notes
- Flash-attention fast paths can materially change throughput and memory use.
- Precision mode affects stability and speed; always pair dtype changes with sanity checks.
- KV/activation memory behavior should be reasoned with explicit tensor shapes.

## Common failure modes
- Ignoring causal mask placement details and misattributing generation errors.
- Confusing parameter shape with runtime activation shape.
- Switching precision without observing gradient/overflow diagnostics.
